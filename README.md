# Хрустекс
## Описание
Приложение по заказу еды при помощи доставки или в самом заведение. Также отслежка доставки или приготовление заказа.

### Структура
- `MobileApp` - исходный код мобильного приложения
- `ServerApi` - серверная часть и база данных
- `Web` - исходный код веб-сайта
